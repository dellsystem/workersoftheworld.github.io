Workers of the world, unite!

workersofthe.world is a novelty domain initially purchased by @dellsystem and whose renewal was kindly funded by @heybenji.

This placeholder site is hosted by GitHub pages. If you have ideas for how to put this domain to better use, and are willing to take on some of that work, open an issue on GitHub.

In the meantime, check out Notes From Below, a socialist journal that uses class composition analysis, and Tech Workers Coalition, a group aiming to build worker power in the tech industry.
